## সম্পর্কে

শেন নং টিসিএম বাংলাদেশের সিলেটে একটি প্রথাগত চীনা চিকিৎসা ক্লিনিকের জন্য তৈরি একটি বাণিজ্যিক ডেমো প্ল্যাটফর্ম। এটি রোগীর অ্যাপয়েন্টমেন্ট বুকিং, বহু-স্তরের পার্টনার রেফারেল সিস্টেম, রোগী-এজেন্ট রিয়েলটাইম চ্যাট, এবং ক্লিনিক অবস্থান ব্যবস্থাপনা পরিচালনা করে — সব কিছু ত্রিভাষিক ইন্টারফেসে মোড়ানো।

## দ্রুত শুরু

```bash
# ফ্রন্টএন্ড
cd frontend && npm install && npm run dev

# ব্যাকএন্ড
cd backend/Shennong.Api && dotnet run
```

ফ্রন্টএন্ড: `http://localhost:5173` · API: `http://localhost:5000`

## প্রযুক্তি সিদ্ধান্ত

### ফ্রন্টএন্ড

**কেন React 19 (Next.js নয়)?**
এটি একটি SPA যার SSR/SSG প্রয়োজন নেই — কন্টেন্ট ডাইনামিক (অ্যাপয়েন্টমেন্ট, চ্যাট, ড্যাশবোর্ড), SEO-চালিত নয়। Next.js আমরা যেমন সার্ভার জটিলতা যোগ করবে তেমন দরকার নেই। সাধারণ React + Vite বিল্ড সহজ রাখে এবং S3/CloudFront-এ ডিপ্লয়মেন্ট তুচ্ছ করে।

**কেন Vite with Rolldown (Webpack/CRA নয়)?**
Vite-এর ডেভ সার্ভার নেটিভ ESM এর মাধ্যমে মিলিসেকেন্ডে শুরু হয়। Rolldown (Rust-ভিত্তিক বান্ডলার) দ্রুত প্রোডাকশন বিল্ড দেয়। CRA মৃত; Webpack এই সাইজের প্রজেক্টের জন্য অতিরিক্ত।

**কেন Tailwind CSS (styled-components/CSS modules নয়)?**
ইউটিলিটি-ফার্স্ট CSS নামকরণ সিদ্ধান্ত দূর করে এবং স্টাইলগুলিকে মার্কআপের সাথে সহ-অবস্থিত রাখে। একজন ডেভেলপার সহ প্রজেক্টের জন্য, হ্রাস করা কনটেক্সট-সুইচিং CSS-in-JS নমনীয়তার চেয়ে বেশি গুরুত্বপূর্ণ।

**কেন Framer Motion?**
পেজ ট্রানজিশন এবং মোডাল অ্যানিমেশন। ক্লিনিক ব্র্যান্ডিং মসৃণ, পলিশড অনুভূতি থেকে উপকৃত হয়। Framer Motion React অ্যানিমেশনের জন্য মানক — ঘোষণামূলক, কর্মক্ষম, ট্রি-শেকিং সহ ছোট বান্ডেল প্রভাব।

**কেন Leaflet (Google Maps নয়)?**
বিনামূল্যে, ওপেন-সোর্স, কোনো API কী প্রয়োজন নেই। ক্লিনিক অবস্থানগুলি স্থির কোঅর্ডিনেট — আমাদের Google-এর রাউটিং বা Places API প্রয়োজন নেই। Leaflet এবং OpenStreetMap টাইলস যথেষ্ট এবং কিছুই খরচ হয় না।

**কেন Recharts?**
অ্যাডমিন ড্যাশবোর্ডের চার্ট প্রয়োজন (ব্যবহারকারী কার্যকলাপ, অ্যাপয়েন্টমেন্ট প্রবণতা)। Recharts React-নেটিভ, রচনামূলক, এবং আমাদের প্রয়োজনীয়তা কভার করে (লাইন, এরিয়া, বার) D3-এর ওজন ছাড়া।

**কেন `@microsoft/signalr` (socket.io নয়)?**
ব্যাকএন্ড ASP.NET Core — SignalR নেটিভ পছন্দ। এটি WebSocket/Long Polling/SSE নেগোসিয়েশন স্বয়ংক্রিয়ভাবে পরিচালনা করে, ASP.NET অথ মিডলওয়্যারের সাথে ইন্টিগ্রেট হয়, এবং স্ট্যাককে Microsoft ইকোসিস্টেমের মধ্যে রাখে। socket.io-এর সার্ভার সাইডে থার্ড-পার্টি ব্রিজ প্রয়োজন হবে।

### ব্যাকএন্ড

**কেন ASP.NET Core 8 (Node.js/Express নয়)?**
আসল ব্যাকএন্ড Node.js ছিল। C#-এ মাইগ্রেশন ইচ্ছাকৃত ছিল:
- **টাইপ সেফটি** — পার্টনার ট্রি রিকারশন এবং ইনভাইট-স্ট্যাটস লজিকে JS-এ সূক্ষ্ম বাগ ছিল যা TypeScript/C# কম্পাইল টাইমে ধরবে
- **async/await ধারাবাহিকতা** — প্রতিটি I/O অপারেশন সঠিকভাবে async, কোনো কলব্যাক হেল নেই
- **EF Core মাইগ্রেশন** — হাতে লেখা ALTER TABLE স্টেটমেন্ট ছাড়া স্কিমা ব্যবস্থাপনা
- **SignalR** — ফার্স্ট-পার্টি রিয়েল-টাইম সমর্থন, socket.io-এর উপর নির্ভরতা নেই
- **রেজুমে মূল্য** — NZ চাকরি বাজারের জন্য এন্টারপ্রাইজ .NET দক্ষতা প্রদর্শন করে

**কেন EF Core (Dapper/raw SQL নয়)?**
Code First মাইগ্রেশন ম্যানুয়াল স্কিমা ব্যবস্থাপনা দূর করে। ডেটা মডেল EF-এর ওভারহেড উপেক্ষাযোগ্য হওয়ার মতো যথেষ্ট সরল। রিকার্সিভ পার্টনার ট্রি কুয়েরির জন্য, আমরা ইন-মেমরি ট্রাভার্সালে ফিরে যাই — EF নেটিভভাবে রিকার্সিভ CTE সমর্থন করে না, এবং ব্যবহারকারীর ভলিউম অপ্টিমাইজেশনের যোগ্য নয়।

**কেন SQLite (ডেভ) → SQL Server (প্রোড)?**
SQLite-এর জিরো সেটআপ প্রয়োজন — `dotnet run` এবং ডাটাবেস বিদ্যমান। EF Core-এর প্রোভাইডার মডেল সুইচকে একটি কানেকশন স্ট্রিং পরিবর্তন করে। প্রোডাকশনের জন্য SQL Server বেছে নেওয়া হয়েছে কারণ এটি Azure-এ বিনামূল্যে (Azure SQL দিয়ে 10 vCores পর্যন্ত), AWS/Azure ইকোসিস্টেমের সাথে ইন্টিগ্রেট হয়, এবং SQLite-এর লকিং সমস্যা ছাড়াই সমবর্তী রাইট সমর্থন করে।

**কেন JWT (সেশন কুকি নয়)?**
ফ্রন্টএন্ড একটি SPA যা API থেকে পৃথকভাবে ডিপ্লয় করা হয়। JWT টোকেন CORS কুকি জটিলতা ছাড়াই ক্রস-অরিজিন কাজ করে। ৭-দিনের মেয়াদ শেষ হওয়া এবং লগইন-এ রিফ্রেশ নিরাপত্তা এবং UX-এর মধ্যে ভারসাম্য বজায় রাখে — অ্যাপয়েন্টমেন্ট বুক করা রোগীদের প্রতিদিন পুনরায় প্রমাণীকরণ করতে হবে না।

**কেন BCrypt (ASP.NET Identity নয়)?**
ASP.NET Identity একটি সম্পূর্ণ ব্যবহারকারী ব্যবস্থাপনা সিস্টেম আনে (নিশ্চিতকরণ, লকআউট, টু-ফ্যাক্টর) যা এই প্রজেক্টের প্রয়োজন নেই। অথ ফ্লো সরল: নিবন্ধন → লগইন → যাচাই। BCrypt আমাদের Identity ফ্রেমওয়ার্কের স্কিমা এবং মিডলওয়্যার ওভারহেড ছাড়াই পাসওয়ার্ড হ্যাশিং দেয়।

### ইনফ্রাস্ট্রাকচার

**কেন Terraform (CDK/Pulumi/Console নয়)?**
পুনরুৎপাদনযোগ্যতার জন্য ইনফ্রাস্ট্রাকচার অ্যাজ কোড অ-আলোচ্যযোগ্য। Terraform ক্লাউড-অ্যাগনস্টিক (আজ AWS, আগামীকাল Azure), সবচেয়ে বড় প্রোভাইডার ইকোসিস্টেম আছে, এবং HCL সিনট্যাক্স পঠনযোগ্য। CDK/Pulumi আপনাকে সাধারণ-উদ্দেশ্য ভাষায় ইনফ্রা লিখতে দেয়, কিন্তু এটি একটি বৈশিষ্ট্য যা আমাদের দরকার নেই — HCL-এর সীমাবদ্ধতা কনফিগারেশনগুলিকে ঘোষণামূলক এবং পর্যালোচনাযোগ্য রাখে।

**কেন AWS ECS Fargate (EC2/Lambda নয়)?**
Fargate আমাদের EC2 ইনস্ট্যান্স পরিচালনা না করেই কন্টেইনার অর্কেস্ট্রেশন দেয়। API একটি দীর্ঘ-চলমান সার্ভার (SignalR WebSocket সংযোগ), তাই Lambda-র কোল্ড-স্টার্ট মডেল ফিট করে না। Fargate-এর পে-পার-রিকোয়েস্ট প্রাইসিং কম-ট্র্যাফিক ক্লিনিক অ্যাপের জন্য কাজ করে।

**কেন Docker Compose নেই?**
ডেভেলপমেন্ট এনভায়রনমেন্টের জিরো বাহ্যিক নির্ভরতা আছে — SQLite এমবেডেড, ফ্রন্টএন্ড একটি স্ট্যাটিক ডেভ সার্ভার। Docker Compose কোনো সুবিধা ছাড়াই জটিলতা যোগ করবে। প্রোডাকশন ডিপ্লয়মেন্ট Terraform → ECS দিয়ে যায়, docker-compose দিয়ে নয়।

## API

JWT অথেনটিকেশন সহ RESTful API।

| মডিউল | এন্ডপয়েন্ট | বিবরণ |
|--------|-----------|-------------|
| অথ | `POST /api/auth/register`, `/login`, `GET /verify` | JWT অথেনটিকেশন |
| ব্যবহারকারী | `GET /api/user/profile/:id`, `POST /bind-inviter` | প্রোফাইল ও রেফারেল বাইন্ডিং |
| অ্যাপয়েন্টমেন্ট | `GET/POST /api/appointments` | বুকিং ও ইতিহাস |
| দর্শনার্থী | `GET/POST/DELETE /api/visitors` | ঘন ঘন দর্শনার্থী ব্যবস্থাপনা |
| পার্টনার | `POST /api/partner/apply`, `GET /tree/:id` | আবেদন ও পদক্রম |
| অ্যাডমিন | `GET /api/admin/all`, `/stats`, `/partners-detailed` | সিস্টেম-ব্যাপী ডেটা |
| এজেন্ট | `GET /api/agent/sessions` | চ্যাট সেশন ব্যবস্থাপনা |
| চ্যাট | `GET /api/chat/history`, `POST /read` + SignalR hub | রিয়েল-টাইম মেসেজিং |

## ডিপ্লয়মেন্ট

`terraform/`-এ Terraform এর মাধ্যমে ইনফ্রাস্ট্রাকচার পরিচালিত। লক্ষ্য: AWS ECS Fargate + RDS + ALB + Route 53।

```bash
cd terraform
terraform init
terraform plan
terraform apply
```
